Node & Edge Schema — Updated

Trimmed to essentials. Edges now cover both inter-file and intra-file relationships.

Node Base — TsNodeBase

id: string
kind: NodeKind
parent: string | null
children: string[]
siblings: string[]

Node Kinds

FolderNode 
// ...TsNodeBase
kind: 'FOLDER'
name: string
path: string
depth: number
FileNode 
// ...TsNodeBase
kind: 'FILE'
name: string
path: string
fileType: 'ts' | 'tsx'
FunctionNode 
// ...TsNodeBase
kind: 'FUNCTION'
params: Param[]
returnType: string | null

// Param = { name: string, type: string }
ClassNode 
// ...TsNodeBase
kind: 'CLASS'
name: string
extends: string | null
implements: string[]
decorators: string[]
constructorParams: Param[]
ImportNode 
// ...TsNodeBase
kind: 'IMPORT'
name: string
source: 'local' | 'package'

Edge Base — TsEdgeBase

id: string
type: 'import' | 'export' | 'call'
source: string     // node id
target: string     // node id

type TsEdge = ImportEdge | ExportEdge | CallEdge

ImportEdge inter-file 
// ...TsEdgeBase
type: 'import'
// FILE → IMPORT
ExportEdge inter-file 
// ...TsEdgeBase
type: 'export'
isReexport: boolean
// FILE → FUNCTION|CLASS|INTERFACE|FILE
CallEdge intra-file 
// ...TsEdgeBase
type: 'call'
// FUNCTION → FUNCTION
// FUNCTION → IMPORT

CallEdge is new — covers intra-file relationships (function calls function, function calls import). Anything else to adjust?
