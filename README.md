
# git-explorer

A small Next.js app that helps you visualize your codebase through git data.

## Setup Instructions

### 1. Clone the repository

```sh
git clone <repo-url>
cd git-explorer
```

### 2. Install dependencies

```sh
npm install
```

### 3. Run the development server

```sh
npm run dev
# Open http://localhost:3000 in your browser
```

### 4. Run tests

```sh
npm test
```

### 5. Build for production

```sh
npm run build
npm start
# Open http://localhost:3000 in your browser
```
